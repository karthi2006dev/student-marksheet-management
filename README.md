# Student Marksheet Management System

A Python-based command-line interface (CLI) application built to manage student records, calculate marks, and generate report cards using Object-Oriented Programming (OOP) principles.

## 🚀 Features
- **Object-Oriented Design:** Clear separation of concerns using `Student` and `Marksheet` classes.
- **Composition:** Implements a "Has-A" relationship where a Marksheet contains a Student object.
- **Data Validation:** Safe integer input handling with error catching for numeric scores.
- **Dynamic Formatting:** Formatted report cards generated directly in the console.

## 🛠️ Tech Stack
- **Language:** Python 3.x
